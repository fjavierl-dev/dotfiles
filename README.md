Keybindings
============


Basic Applications

| Shortcut            | Action                              |
|---------------------|--------------------------------------|
| SUPER + ENTER       | Open terminal (Kitty)               |
| SUPER + B           | Open Brave browser                  |
| SUPER + Q           | Close active window                 |
| SUPER + R           | Reload Hyprland configuration       |
| SUPER + D           | Open Rofi application launcher      |

---

Focus Navigation (Directional)

| Shortcut                  | Action              |
|---------------------------|--------------------|
| SUPER + ALT + ←           | Move focus left     |
| SUPER + ALT + →           | Move focus right    |
| SUPER + ALT + ↑           | Move focus up       |
| SUPER + ALT + ↓           | Move focus down     |

---

Persistent Workspaces

Workspaces **1–4** are configured as persistent.

| Shortcut      | Action                    |
|--------------|--------------------------|
| SUPER + 1    | Switch to workspace 1    |
| SUPER + 2    | Switch to workspace 2    |
| SUPER + 3    | Switch to workspace 3    |
| SUPER + 4    | Switch to workspace 4    |

---

Move Windows (Floating or Tiled)

| Shortcut                    | Action             |
|-----------------------------|-------------------|
| SUPER + CTRL + ←            | Move window left  |
| SUPER + CTRL + →            | Move window right |
| SUPER + CTRL + ↑            | Move window up    |
| SUPER + CTRL + ↓            | Move window down  |

---

Focus Switching (Alternative Method)

| Shortcut                    | Action              |
|-----------------------------|--------------------|
| SUPER + SHIFT + ←           | Focus left window   |
| SUPER + SHIFT + →           | Focus right window  |
| SUPER + SHIFT + ↑           | Focus upper window  |
| SUPER + SHIFT + ↓           | Focus lower window  |
